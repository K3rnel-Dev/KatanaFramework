## Setup

#### Clone the repository
```shell
git clone https://github.com/k3rnel-dev/katana-framework
cd katana-framework
```

#### Install requirements
```shell
chmod +x install.sh
./install.sh
```

## Usage

#### Search by username
```python
python katana.py
```
